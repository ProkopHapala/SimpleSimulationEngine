# NavalBattle

Simulation of a battleship with multiple cannon turrets. 2D side-view naval combat with realistic buoyancy, turret rotation, and projectile ballistics.

## Files

- **NavalBattle_main.cpp** — main application: ship control, turret aiming, firing, naval combat
- **Battleship.cpp / .h** — battleship model: hull, turrets, buoyancy, cannon firing
- **CMakeLists.txt** — build target: `NavalBattle_main`
- **data/** — ship configuration data
