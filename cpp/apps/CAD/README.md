# CAD

Simple 2D CAD (Computer-Aided Design) tool and analytical geometry demonstration. Interactive drawing and manipulation of 2D geometric primitives with constraint solving.

## Files

- **cad2d.cpp** — main application: 2D CAD with drawing, selection, and geometric operations
- **CAD2D.h** — CAD data structures and operations interface
- **CMakeLists.txt** — build target: `cad2d`
- **python/** — Python helper scripts
