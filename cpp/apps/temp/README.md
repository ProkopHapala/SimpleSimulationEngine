# temp — Archived/Experimental Applications

Legacy and experimental applications not in the active CMake build. Kept for reference.

## Subdirectories

- **Fortress/** — fortress/castle simulation (headers only, incomplete)
- **NBodyWorld/** — 2D N-body gravitational simulation with `NBodyWorld2D` class. Has its own CMakeLists.txt.
- **SailWar_h_bak/** — backup of SailWar headers (18 files)
- **Ship2D/** — 2D ship simulation with Code::Blocks project files (`.cbp`). Legacy, standalone makefile.
- **SpaceCraft/** — early spacecraft model headers (9 files in `include/`)
